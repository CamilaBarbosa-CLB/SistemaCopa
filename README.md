# Sistema Copa

Projeto desenvolvido para fins acadêmicos com o objetivo de aplicar conceitos de Programação Orientada a Objetos (POO) utilizando a linguagem Java.

## Funcionalidades

* Cadastro de seleções
* Consulta de seleções
* Atualização de dados das seleções
* Exclusão de seleções
* Cadastro e gerenciamento de jogos

## Estrutura do Projeto

* **controller**: controle das operações do sistema
* **dao**: acesso aos dados
* **model**: classes que representam as entidades do sistema
* **view**: execução e testes das funcionalidades
* **factory**: gerenciamento da conexão com o banco de dados

## Tecnologias Utilizadas

* Java
* Eclipse IDE
* Programação Orientada a Objetos (POO)

## Objetivo

O projeto tem como finalidade praticar conceitos de desenvolvimento em Java, organização em camadas e manipulação de dados em um sistema de gerenciamento de uma copa de futebol.
